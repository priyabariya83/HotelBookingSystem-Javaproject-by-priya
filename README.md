# HotelBookingSystem-Javaproject-by-priya
A console-based Java application that allows hotel staff to manage room bookings. It includes features such as room addition, booking with customer details, cancellation, and bill calculation with 18% GST. The system also uses file handling to save and load booking data, and includes basic admin authentication.
